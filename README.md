# Meu Portfólio Pessoal ✨

Bem-vindo ao código fonte do meu portfólio online! Este projeto foi desenvolvido com o objetivo de apresentar minhas habilidades, projetos e certificados como desenvolvedor Front-end e Back-end.

O site é uma Single Page Application (SPA) estática, construída primariamente com tecnologias web fundamentais, com conteúdo dinâmico carregado via JavaScript para flexibilidade na gestão de projetos e certificados.

Acesse a versão publicada aqui: [Link para o seu GitHub Pages ou domínio personalizado]

## Tecnologias Utilizadas

Este projeto utiliza as seguintes tecnologias e bibliotecas:

* **HTML5:** Estrutura semântica da página.
* **CSS3:** Estilização e responsividade.
* **JavaScript:** Manipulação do DOM, carregamento dinâmico de conteúdo, interatividade (menu mobile, modal, scroll spy, animações).
* **Boxicons:** Biblioteca de ícones utilizados em todo o site.
* **Font Awesome:** Utilizado para alguns ícones sociais.
* **Google Fonts:** Para tipografia personalizada (`Poppins` e `Roboto`).

## Funcionalidades Principais

* **Design Responsivo:** O layout se adapta a diferentes tamanhos de tela (desktop, tablet, mobile).
* **Navegação Fixa e Suave:** Menu de navegação fixo no topo com scroll suave para as seções.
* **Scroll Spy:** Destaque automático do item do menu correspondente à seção visível na tela.
* **Menu Mobile (Hamburger):** Navegação otimizada para dispositivos móveis.
* **Carregamento Dinâmico:** Projetos e Certificados são carregados dinamicamente a partir de dados em JavaScript, facilitando a atualização do conteúdo sem alterar o HTML.
* **Visualização de Habilidades:** Apresentação das Hard Skills com barras de progresso animadas ao entrar na tela.
* **Modal/Lightbox de Certificados:** Visualização ampliada dos certificados de imagem em um modal.
* **Links de Credencial/Verificação:** Para certificados relevantes, links diretos para as credenciais online ou páginas de verificação.
* **Seção de Contato:** Links diretos para email, LinkedIn e GitHub.

## Estrutura do Projeto

O projeto segue uma estrutura de arquivos simples:


.
├── index.html
├── styles.css
└── scripts.js
└── README.md
└── ... (outros arquivos como imagens, etc.)


* `index.html`: Contém a estrutura principal da página e define as seções.
* `styles.css`: Contém todas as regras de estilo CSS para o layout e aparência.
* `scripts.js`: Contém o código JavaScript para interatividade, carregamento dinâmico e outras funcionalidades.

# My Personal Portfolio ✨ (EN)

Welcome to the source code of my personal online portfolio! This project was developed with the goal of showcasing my skills, projects, and certificates as a Front-end and Back-end developer.

The website is a static Single Page Application (SPA), built primarily with fundamental web technologies, featuring dynamic content loading via JavaScript for flexibility in managing projects and certificates.

Access the published version here: [Link to your GitHub Pages or custom domain]

## Technologies Used

This project utilizes the following technologies and libraries:

* **HTML5:** Semantic page structure.
* **CSS3:** Styling and responsiveness.
* **JavaScript:** DOM manipulation, dynamic content loading, interactivity (mobile menu, modal, scroll spy, animations).
* **Boxicons:** Icon library used throughout the site.
* **Font Awesome:** Used for some social icons.
* **Google Fonts:** For custom typography (`Poppins` and `Roboto`).

## Key Features

* **Responsive Design:** The layout adapts to different screen sizes (desktop, tablet, mobile).
* **Fixed and Smooth Navigation:** Fixed navigation bar at the top with smooth scrolling to sections.
* **Scroll Spy:** Automatic highlighting of the menu item corresponding to the visible section on screen.
* **Mobile Menu (Hamburger):** Optimized navigation for mobile devices.
* **Dynamic Loading:** Projects and Certificates are loaded dynamically from JavaScript data, making content updates easy without altering the HTML.
* **Skills Visualization:** Presentation of Hard Skills with animated progress bars that trigger upon entering the viewport.
* **Certificate Modal/Lightbox:** Enlarged viewing of image-based certificates in a modal window.
* **Credential/Verification Links:** For relevant certificates, direct links to online credentials or verification pages.
* **Contact Section:** Direct links for email, LinkedIn, and GitHub.

## Project Structure

The project follows a simple file structure:


.
├── index.html
├── styles.css
└── scripts.js
└── README.md
└── ... (other files like images, etc.)


* `index.html`: Contains the main page structure and defines the sections.
* `styles.css`: Contains all the CSS style rules for layout and appearance.
* `scripts.js`: Contains the JavaScript code for interactivity, dynamic loading, and other features.
